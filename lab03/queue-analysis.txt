
NOTE:
- Your time complexities should be expressed in big-O notation and be in
  terms of n, where n is the size (i.e., number of items) of the queue

=========
ListQueue
=========

-------------+----------------------------------------------------------
             |                    time complexity
  operation  +-----------------+------------------+---------------------
             |    best case    |    worst case    | amortised (optional)
-------------+-----------------+------------------+---------------------
             |                 |                  |
   enqueue   |        O(1)     |       O(1)       |  O(1)
             |                 |                  |
-------------+-----------------+------------------+---------------------
             |                 |                  |
   dequeue   |      o(1)        |     O(1)        | O(1)
             |                 |                  |
-------------+-----------------+------------------+---------------------
